一、十大词性（词类）
1、动词（Verb）
    1.1、谓语动词（Predicate Verb）
        1.1.1、谓语动词的分类
            （1）不及物动词
            （2）单及物动词
            （3）双及物动词
            （4）复杂及物动词
            （5）系动词
        1.1.2、谓语动词的三种功能
            （1）表示动作时间（Tense）
                ① 现在：动作在现在发生。
                ② 过去：动作在过去发生。
                ③ 将来：动作在将来发生。
                ④ 过去将来：针对于过去的某一个时间点来说，动作在“将来”发生，这里的“将来”相对于“现在”这个时间参照点来说，可以是过去，可以是现在，也可以是将来。
            （2）表示动作状态（Aspect）
                ① 一般：动作的发生没有具体说明或者强调状态，通常只是在描述一种事实。
                ② 完成：动作的发生已经完成了。
                ③ 进行：动作的发生还在进行中。
                ④ 完成进行：动作的一部分完成了，但是动作还在进行中。
            （3）表示动作语气（Mood）
                ① 虚拟语气：
                ② 陈述语气：
                ③ 祈使语气：
        1.1.3、动作时间和动作状态可以进行任意排列组合
            （1）现在：
                ① 一般现在时态：兔子吃胡萝卜。
                    为：动词原形 + s（如果是第三人称单数）
                ② 现在完成时态：兔子现在已经吃完了胡萝卜。
                    为：助动词have的变位(have,has) + 动词的过去分词
                ③ 现在进行时态：兔子现在正在吃胡萝卜。
                    为：助动词be的变位(am,is,are) + 动词的现在分词
                ④ 现在完成进行时态：兔子现在已经吃完了一些胡萝卜，还要继续吃。
                    为：助动词have的变位(have,has) + been + 动词的现在分词
            （2）过去：
                ① 一般过去时态：兔子过去吃了胡萝卜。
                    为：动词过去式
                ② 过去完成时态：兔子昨天下午三点已经吃完了胡萝卜。
                    为：助动词have的变位(had) + 动词的过去分词
                ③ 过去进行时态：兔子昨天下午三点正在吃胡萝卜。
                    为：助动词be的变位(was,were) + 动词的现在分词
                ④ 过去完成进行时态：兔子昨天下午三点已经吃完了一些胡萝卜，还要继续吃。
                    为：助动词have的变位(had) + been + 动词的现在分词
            （3）将来：
                ① 一般将来时态：兔子将来会吃胡萝卜。
                    为：will + 动词原形
                ② 将来完成时态：兔子明天下午三点会已经吃完了胡萝卜。
                    为：will + have + 动词的过去分词
                ③ 将来进行时态：兔子明天下午三点会正在吃胡萝卜。
                    为：will + be + 动词的现在分词
                ④ 将来完成进行时态：兔子明天下午三点会已经吃完了一些胡萝卜，还要继续吃。
                    为：will + have + been + 动词的现在分词
            （4）过去将来：
                ① 一般过去将来时态：我之前就知道兔子会去吃胡萝卜。
                    为：
                        would + 动词原形 
                    或者     
                        助动词be的变位(was,were) + going + to + 动词原型
                ② 过去将来完成时态：我之前就知道兔子会在明天下午三点会已经吃完了胡萝卜。
                    为：would + have + 动词的过去分词
                ③ 过去将来进行时态：我之前就知道兔子会在明天下午三点会正在吃胡萝卜。
                    为：would + be + 动词的现在分词
                ④ 过去将来完成进行时态：我之前就知道兔子会在明天下午三点会已经吃完了一些胡萝卜，还要继续吃。
                    为：would + have + been + 动词的现在分词
        1.1.4、补充
            （1）完成以上三种功能并不是谓语动词仅凭本身就可以实现，还是需要用到助动词。
            （2）动词有原型、第三人称单数、过去式、过去分词、现在分词等五种变位形式，但是助动词或者情态动词并不一定有全部的变位形式。
    1.2、非谓语动词（Non-finite Verb）可以充当除谓语动词外的所有句子成分，取代几乎所有的从句，简化句子。

二、八种基本句型
1、可以独立完成的动作，主语 + 不及物动词
2、有1个动作的承受者，主语 + 单及物动词 + 宾语
3、有2个动作的承受者，主语 + 双及物动词 + 间接宾语 + 直接宾语
4、只有1个动作承受者（但需要补充），主语 + 复杂及物动词 + 宾语 + （宾语）补语
5、非“动作”，主语 + 系动词 + （主语）补语
6、“there be”句型，可以理解为第5种句型的倒装句型。
7、主语 + 谓语动词 + 状语，可以理解为第1种句型的延伸句型。
8、主语 + 谓语动词 + 宾语 + 状语，可以理解为第四种句型的延伸句型。
注：
    1、“主语补语”在中国的英语教学中又称为“表语”。
    2、“谓语”和“谓语动词”属于包含关系，句子主语后面余下的句子部分就是谓语，谓语动词只是谓语的一部分。
